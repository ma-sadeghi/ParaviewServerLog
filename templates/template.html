How to connect to the Paraview servers

Whether you're using Linux or Windows, open up a terminal (or cmd in Windows) and run the following command: 

ssh anaconda@pmealsrv1.uwaterloo.ca -L 11111:localhost:11111

Note 1: Make sure you're connected to UW's VPN.

Note 2: Make sure to choose a port number that is available (see table below).
